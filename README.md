# Melanoma-Detection
# 🩺 Melanoma Skin Cancer Detection using YOLOv8

A desktop application powered by **YOLOv8** and **CustomTkinter** to detect Melanoma skin cancer from medical images. This project leverages deep learning to classify images into **Melanoma** or **Not Melanoma** with high accuracy.

## ✨ Features
- 🖱️ User-friendly GUI with Dark/Light theme toggle
- 📸 Upload JPG, PNG, BMP, or TIFF images
- ⚡ Real-time classification using YOLOv8
- 📊 Displays diagnosis result with confidence percentage
- 🚀 Standalone Windows executable (available in Releases)
